Summary: Latitude vs. X

The purpose of this project was to analyze trends in weather data based on latitude and to show an understanding of HTML, CSS, and the Bootstrap CSS tool.

After creating the combined data, the Pandas library was used to analyze the data. A set of plots was generated using Matplotlib that display relations between Temperature, Humidity, Cloudiness, and Windspeed vas latitude. The source data for the graphs is provided in this site along with detailed explanations of each graph.

 
This site contains these individual graphs along with descriptions, a comparisons page, and a data page with a responsive table with the complete source data used to conduct the analysis.
