Summary: Latitude vs. X

The purpose of this project was to analyze trends in weather data based on latitude and to show an understanding of HTML, CSS, and the Bootstrap CSS tool.

After assembling the dataset, a Pandas dataframe was created and a series of graphs were generated using Matplotlib that display Max Temperature, Humidity, Cloudiness, or Windspeed on the Y-Axis and Longitude on the X-Axis. The source data for the graphs is provided in this site along with detailed explanations of each graph.

 

This site contains these individual graphs along with descriptions, a comparisons page, and a data page with a responsive table containing the data used in the visualizations.
